## BUILD

### DOCKER RUN

```\
docker run -d --rm \
--name gophish \
-p 443:443 \
-p 80:80 \
-p 127.0.0.1:3333:3333 \
alexandreoda/gophish
```

link to Gophish <https://127.0.0.1:3333/login>

ID        = admin

PASSWORD  = gophish

### DOCKER COMPOSE

```yml
version: "2.0"

services:
  gophish:
    container_name: gophish
    image: alexandreoda/gophish
    privileged: false
    ports:
      - "443"
      - "80"
      - "3333"
```
